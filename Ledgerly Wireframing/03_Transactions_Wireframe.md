# Transactions Wireframe

```
+-----------------------------+
|      Transactions           |
|-----------------------------|
| [Filter]   [Search]         |
|-----------------------------|
| 04/27  $50  Grocery         |
| 04/26  $120 Rent            |
| 04/25  $15  Coffee          |
| ...                         |
|-----------------------------|
| [ + Add Transaction ]       |
+-----------------------------+

Add/Edit Transaction Form:
+-----------------------------+
| Amount:  [      ]           |
| Date:    [      ]           |
| Category:[      ]           |
| Notes:   [      ]           |
| [Attach Receipt]            |
| [Save]   [Cancel]           |
+-----------------------------+
```

- List view with filter/search
- Add/edit form for transactions
- Support for attachments and notes
